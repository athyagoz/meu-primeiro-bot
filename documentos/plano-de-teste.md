# Plano de Teste - Projeto de Gestão de Funcionários

## 1. Objetivo
Validar as principais funcionalidades do sistema de cadastro de funcionários, garantindo que o fluxo de usuário esteja de acordo com o protótipo fornecido.

## 2. Escopo 
- **Interface:** Comparação visual com o protótipo.
- **Funcionalidades:** Criação, edição e exclusão de funcionários.
- **Validação:** Verificação de campos obrigatórios e formato de CPF.
- **Navegação:** Teste dos links do menu lateral.

## 3. Ambiente de Teste
- **Navegador:** Google Chrome.
- **Dispositivo:** Computador (Desktop).
- **URL do Site:** analista-teste.seatecnologia.com.br

## 4. Ferramentas Utilizadas
- **GitHub:** Para documentação e controle de versão.
- **Python:** Para lógica de automação de testes.
